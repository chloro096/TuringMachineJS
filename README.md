# TuringMachineJS

# チューリングマシンとは？

1936 年に、アラン・チューリングが考えた架空の機械です。
構造はシンプルで、文字が一つ書き込める「マス目」がたくさん並んだ 1 次元の記憶テープと、マス目を読み書きするヘッド、読み取った内容に基づいて処理を行う制御部の 3 つからなります。
制御部の処理に基づいて、テープを左右に動かしながら文字を読み書きすることで、様々な計算を行うことができます。

## 何がすごいの？

チューリングマシンの偉大さは、その汎用性にあるといえます。
結論から言うと、現在のコンピュータで計算できることは、どんなことでもチューリングマシンで再現できます。
つまり、チューリングマシンに計算できないことは、少なくとも今の人類には計算できないということです。
さらに言うと、今後量子コンピュータやその他のどんなコンピュータが登場したとしても、チューリングマシンに計算できないことが計算できるようになることはないだろうと考えられているそうです。
この意味で、チューリングマシンで計算可能なこととは、すなわちこの世界で「計算可能なこと」だといえます。
この「チューリングマシンによって計算可能なことを計算可能とする」という前提は、チャーチ=チューリングのテーゼと呼ばれ、最も重要なアイデアの一つとして現在の計算機科学の根幹を担っています。
